# Login, Registration & Dashboard System – User Manual

## Pages

### 1. Login Page (`login.html`)
- Enter your **username** and **password**.
- Click **Login**.
- After entering credentials, you are redirected to the **Dashboard**.
- Dashboard is unaccesable if not logged in.
- You can choose to register if no account.

### 2. Register Page (`register.html`)
- Fill in all fields:
  - First Name
  - Last Name
  - Username
  - Email
  - Password
  - Confirm Password
- Password and Confirm Password must match.
- Click **Register** to create an account.
- After successful registration, you are redirected to the login page.

### 3. Dashboard Page (`dashboard.html`)
- Accessible only if logged in.
- Displays a table of user data.
- Click **Logout** to end the session and return to the login page.

---

## How to Use

1. Open `login.html` in a web browser.
2. If you don’t have an account, click **Register**.
3. Complete the registration form.
4. Log in with your credentials.
5. View the dashboard and data table.
6. Click **Logout** to exit.
